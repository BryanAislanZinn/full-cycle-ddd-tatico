# full-cycle-ddd-tatico
Repositório do Curso de DDD Tático da Full Cycle

## Instruções

- execute `git clone https://github.com/BryanAislanZinn/full-cycle-ddd-tatico.git` para clonar o projeto
- execute `npm install` para criar o projeto (recomendado executar pelo menos com o node 20);
- execute `npm test` para executar os testes;
